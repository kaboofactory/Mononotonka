# 🎮 Mononotonka

**MonoGameをもっとラクに、もっと楽しく。**

Mononotonkaは、MonoGame向けの2Dゲーム開発に特化したラッパーライブラリです。  
面倒なボイラープレートをぶっ飛ばして、ゲーム開発の本質に集中できます。

---

## ✨ 特徴

| 機能 | 説明 |
|------|------|
| 🎬 **シーン管理** | `Ton.Scene.Change()` 一発でシーン遷移 |
| 🖼️ **描画システム** | スプライト、テキスト、プリミティブ描画をシンプルに |
| 🎵 **サウンド** | BGM/SE管理、フェード、ループ制御 |
| 🎮 **入力** | キーボード・ゲームパッド・マウスを統一API化 |
| 💬 **メッセージ** | ADV風テキスト表示システム内蔵 |
| 📋 **メニュー** | カーソル操作可能なメニューUI |
| 🌟 **パーティクル** | お手軽エフェクト生成 |
| 💾 **セーブ/ロード** | ストレージ管理も楽々 |

## 📁 プロジェクト構成

```
Mononotonka/
├── mononotonka/     # 📦 ライブラリ本体
├── Content/         # 🎨 MGCBコンテンツ
├── font/            # 🔤 SpriteFontファイル
├── shader/          # ✨ シェーダー
├── sample_assets/   # 🎭 サンプル用素材
├── manual/          # 📖 ドキュメント
└── SampleScene*.cs  # 🎮 サンプルシーン
```

## ⚠️ 注意事項

- **フォントファイル (TTF/OTF)** は同梱されていません
- `font/` 内の `.spritefont` を使用するには、対応するフォントを別途配置してください
- 詳細は `font/readme.txt` を参照
