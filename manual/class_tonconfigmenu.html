Class: TonConfigMenu

簡易的な設定メニュー（フルスクリーン切替、Master/BGM/SE音量、メッセージ速度、バックグラウンド時ミュート）を表示・制御します。

Methods

void Open()

メニューを開きます。

void Close()

メニューを閉じます。

bool IsOpen()

メニューが開いているかどうかを返します。

void Update()

メニューの入力を処理・更新します。Ton.Updateから自動的に呼び出されます。

void Draw()

メニューを描画します。Ton.Drawから自動的に呼び出されます。

Sample Code

// Open Config Menu (e.g. from Title Screen or Pause)
Ton.ConfigMenu.Open();

// Checking state
if (Ton.ConfigMenu.IsOpen())
{
    // Pause Game Logic
}

// Close manually if needed
Ton.ConfigMenu.Close();

// Update/Draw are called automatically by Ton system
